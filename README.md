# Open-Source Mindfulness App Feature Comparison

Proyek ini bertujuan untuk mengumpulkan, membandingkan, dan menyediakan data komprehensif mengenai fitur-fitur yang ditawarkan oleh aplikasi mindfulness dan meditasi populer (baik yang berbayar maupun yang open-source). Data yang dikumpulkan meliputi:
* Fitur Kunci (misalnya, sesi tidur, musik, sesi bergerak)
* Struktur Harga
* Aspek Aksesibilitas

Data ini tersedia dalam format CSV/JSON di direktori `/data`. Kami menyambut kontribusi dan pembaruan data!
